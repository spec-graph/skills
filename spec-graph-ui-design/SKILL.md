# spec-graph-ui-design

## Required Sections

1. **User flows**: Step-by-step interactions, decision points, error/empty/loading states
2. **Component tree**: Atomic → Molecular → Organism → Page
3. **Design tokens**: Color, typography, spacing, border, shadow
4. **Accessibility**: WCAG AA (contrast ≥ 4.5:1), keyboard nav, screen reader, focus management
5. **Responsive design**: Mobile (320-768px), Tablet (768-1024px), Desktop (1024+)

## Conservative

- Design mobile-first
- Design for keyboard from start
- Always include error/empty/loading states
